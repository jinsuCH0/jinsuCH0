# 🎧 지뭐듣(지금 뭐 듣지?)
#### "지뭐듣'은 위치 기반 사용자 간 음악 추천 어플입니다. <br><br><br>

## 🤔 <I>Why?</I> <- 프로젝트 개요
> 매일 다니는 골목 길을 걸을 때마다 듣는, 퇴근 길에, 여행지에서... 처럼 모두들 장소와 관련된 음악이 한 곡쯤은 있을 것 입니다.<br><br> 
> <I>'지뭐듣'</I> 은 유저끼리 단순히 음악만을 추천하는 것을 넘어,  
> 위치와 음악을 "감정"으로, 유저와 유저를 "공감"으로 연결하기 위해 만들어졌습니다.
